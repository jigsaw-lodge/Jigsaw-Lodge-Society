FULL SECOND LIFE TESTING PACKAGE
2026-04-09

PURPOSE
-------
This file is the plain-English manifest for the LSL package needed to test
the full MMO against the current backend.

Use it when:
- exporting the missing in-world scripts into the repo
- asking ChatGPT to help write or repair one object at a time
- deciding which SL objects are required for a real end-to-end test

Current truth:
- the backend is authoritative
- LSL is input/output only
- signed requests are preferred
- shared-token mode is a migration fallback only
- only two HUD scripts are currently checked into the repo
- public kiosk behavior is being consolidated into one marketplace object
- the wearable HUD is expected to expose a real marketplace menu

AUTH AND CONFIG FOR EVERY SCRIPT
--------------------------------
Preferred auth mode:
- timestamp
- request_id
- signature

Fallback auth mode:
- token

Shared config values:
- API_EVENT_URL = https://jigsawlodgesociety.com/api/event
- API_BASE_URL = https://api.jigsawlodgesociety.com/api
- WORLD_URL = https://api.jigsawlodgesociety.com/api/world
- RELAY_URL = wss://ws.jigsawlodgesociety.com
- FALLBACK_HTTP = http://89.167.94.250:3000

REQUEST RULES
-------------
- every object should send avatar
- signed requests should also send timestamp, request_id, and signature
- objects should throttle and retry with backoff
- do not let public objects hold ADMIN_TOKEN
- use isolated test zones, not 0:0
